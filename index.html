Frequently Asked Questions

What is this page about?

This page is a demonstration of various interactive elements created with JavaScript.

How does the theme toggle work?

The theme toggle button uses JavaScript to add or remove a "dark-mode" class to the body of the page, which then changes the color scheme via CSS.

How does the FAQ section work?

The FAQ section uses JavaScript to listen for clicks on the questions. When a question is clicked, it toggles the visibility of the answer.
